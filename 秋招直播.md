  + 编程题
    + 给定一个图，求其是否是完全多部图（可以将点划分为若干个集合，每个集合内的点两两不直接连边，每两个不属于同一个集合的点两两连边）
      + 对图取反，转化为求图的所有连通块是否为完全图
    + 给定n个三元整数组（n取到500000）`<ai,bi,ci>`，问其中有多少个满足，在n个三元组中存在一个使得其所有元素均大于它
      + 如果是二元组的话，显然对a降序排序之后维护一个集合保存当前扫过的b，然后从左到右扫一遍不断判断集合中是否有比当前b大的数，并将b分批更新进集合即可
        + 按a分批更新，所有a相同的二元组需要同时更新进集合中
      + 于是显然可以想到用一个线段树，以b为区间下标，维护c的最大值，代替上面说的集合
        + 判断的步骤则判断`[bi+1,max]`中是否有比当前`ci`更大的数（取其最大值下来比较即可）
        + 更新的话则把`<bi,ci>`更新进线段树即可
        + 由于b的取值范围过大，需要将其离散化
